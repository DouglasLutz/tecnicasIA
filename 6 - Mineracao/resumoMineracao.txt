    MINERAÇÃO DE DADOS
        - banco de dados 
        - texto (web)
        - texto (plain)

O que é mineração de dados?
    - ferramenta de descoberta de conhecimento (padrões, relações, tendências, ...)
    - faz parte da área de Descoberta de Conhecimento KDD

Quando usar?
    - muitos dados não organizados

Processo de minerar:
    - definir a base de dados
    - limpeza da base  (cria o armazem de dados)
    - definir o algoritmo de mineração: associação, aglomerar, classificação, série temporais
